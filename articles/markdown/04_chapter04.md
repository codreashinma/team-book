# 第4章　商店街のSOS

---

　根津大地は、話を最後まで聞かなかった。

「暗号資産？　悪いが、そういうのは間に合ってる」

　蓮がノートパソコンの画面を見せようとした手が、空中で止まった。ねづ珈琲のカウンター。朝十時、開店直後。客はまだ誰もいない。コーヒーの香りだけが、二人の間に漂っている。

「根津さん、まだ何も——」

「聞かなくても分かるよ。仮想通貨で決済がどうの、手数料がタダになるだの、そういう話だろ。去年もその手の営業が二組来た。どっちもスーツ着た若いのが横文字並べて、結局何を言いたいのか分からなかった」

　根津は奥のシンクに向かい、カップを洗い始めた。水音が会話を打ち切る壁になる。

「暗号資産じゃないんです」

　蓮は根津の背中に向かって言った。水音は止まらない。

「J-Coinは日本円と同じ価値のステーブルコインで——」

「横文字」

「……すみません。えっと、一コインが一円と同じ値段で——」

「ビットコインみたいに暴落するやつだろ」

「しません。法律で守られていて——」

　根津が振り返った。手には濡れた布巾。その目は冷たくはなかったが、温かくもなかった。真剣に聞く気はないが、蓮を傷つける気もない。三十年間客商売をしてきた人間特有の、絶妙な距離感だった。

「蓮。お前のことは好きだよ。腕のいいエンジニアだってのも分かる。でもな、俺が信用するのは、目の前で手渡しできるものだけだ。お札か、硬貨か、せめてカード。画面の中の数字を『お金です』って言われても、俺にはピンとこないんだよ」

　蓮は口を開きかけて、閉じた。

　反論はできた。技術的にはいくらでも。J-Coinの法的根拠、信託保全の仕組み、PolygonのL2としての処理速度。ここ数日で学んだ知識は、頭の中にぎっしり詰まっている。

　でもそれは全部、蓮の言語だった。

　根津の言語ではない。

「……もう少し、考えさせてください」

「ああ。コーヒーは飲んでいけ」

　根津はブレンドを淹れ始めた。なにごともなかったかのように。蓮はカウンターに座り、目の前に置かれたカップに手を伸ばした。湯気が立ち上る。苦い。今日は、いつもより苦く感じた。

---

　ねづ珈琲を出た蓮は、商店街を歩いた。

　歩くつもりはなかった。自宅に帰ってコードを書くつもりだった。でも足が、勝手に商店街のアーケードに向かっていた。

　根津商店街。正式名称は「根津銀座商店街」。全長約二百メートル。かつては百軒以上の店が軒を連ねていたが、今は四分の三近くがシャッターを下ろしている。残っている店も、どこか疲れた顔をしている——と思うのは、蓮の気のせいではないだろう。

　八百屋の前を通りかかったとき、店先にいた女性店主と目が合った。蓮がねづ珈琲の常連であることを知っているらしく、「あら、根津さんとこの」と声をかけてきた。

「お兄さん、パソコンの仕事してるんだって？　うちのレジ、調子悪いんだけど見てくれない？」

　蓮は苦笑しながらレジを覗いた。古いPOSレジ。画面の端にキャッシュレス決済端末が接続されている。

「これ、手数料けっこう取られません？」

　言ってから、しまった、と思った。他人の懐事情を聞くような質問だ。でも八百屋の女性——木村さんと名札に書いてある——は、遠慮なく答えた。

「取られるよぉ。三パーセントちょっと。キャベツ一玉二百五十円で八円。八円って言ったら笑うかい？　でもね、一日の売上が十万超える日もあるから、そうすると三千円以上よ。週に二万、月に八万。八万あったら冷蔵ケースの修理費が出るのに」

　木村さんは笑いながら言った。笑っているが、目は笑っていなかった。

　蓮は商店街をさらに歩いた。

　煎餅屋。和菓子屋。文具店。金物屋。寿司屋。花屋。どの店の前でも立ち止まり、世間話のふりをして同じ質問を投げた。「キャッシュレス決済、使ってますか？」

　答えは、驚くほど似通っていた。

「手数料が高い。でもやめたら客が減る」

「カード会社から入金されるまで一ヶ月かかる。資金繰りがきつい」

「導入したけど、機械の使い方がよく分からない。サポートに電話しても繋がらない」

「無料キャンペーンで始めたのに、気づいたら有料。詐欺みたいだよ」

　蓮はスマートフォンのメモアプリに、店名と声を書き留めていった。指が追いつかないほど、言葉が溢れてくる。

　六軒回って、蓮は商店街の端のベンチに座った。メモを見返す。どの店も同じ構造的な問題に苦しんでいた。

　**手数料が高い。でもキャッシュレスをやめる選択肢はない。**

　なぜ手数料は高いのか。蓮はエンジニアの頭で考えた。

　クレジットカード決済の場合、お金は客から店に直接届かない。客がカードを使うと、まず**イシュア**（カード発行会社）がカード会員の利用を承認する。次に**アクワイアラ**（加盟店管理会社）が店舗側の処理をする。その間に**カードブランド**（VisaやMastercard）のネットワークを経由する。さらに多くの場合、店舗と契約しているのはカード会社そのものではなく**決済代行業者**だ。

　四つのプレイヤー。それぞれが正当なサービスを提供し、それぞれが手数料を取る。

　客が五百円のコーヒーをカードで払う。その五百円は、客の口座→イシュア→カードブランド→アクワイアラ→決済代行→店舗口座という経路をたどる。各地点で少しずつ引かれ、根津の手元に届くのは四百八十三円。差額の十七円は、この経路の「通行料」だ。

　蓮はメモアプリに図を描いた。

```
客 →[イシュア]→[カードブランド]→[アクワイアラ]→[決済代行]→ 店
     ↑各段階で手数料が発生（合計3〜4%）
```

　そして、その下にもう一つの図を描いた。

```
客 →[ブロックチェーン]→ 店
     ↑ガス代のみ（1円未満）
```

　二つの図の違いは明白だった。仲介者の数が違う。四つと一つ。いや、ブロックチェーンの場合、「仲介者」という概念自体が違う。ブロックチェーンは特定の企業ではなく、分散されたネットワークだ。誰かの利益のために手数料を取る主体が存在しない。必要なのは、ネットワークを維持するための最小限のコスト——Polygonなら一円未満のガス代——だけだ。

　蓮はベンチから立ち上がった。

　技術的には可能だ。J-CoinのSDKを使えば、QRコード決済アプリは作れる。蓮の得意なNext.jsとReactで、フロントエンドは構築できる。SDKがウォレット接続とトークン送金を抽象化してくれるなら、ブロックチェーン歴一週間の蓮でも手が届く範囲だ。

　問題は技術じゃない。

　根津の顔が浮かんだ。「画面の中の数字を『お金です』って言われても、俺にはピンとこないんだよ」。

　根津は技術が分からないのではない。技術を**信頼できない**のだ。

　三十年間、手でコーヒーを淹れてきた人間にとって、「目に見えないもの」は信頼の対象にならない。お札には触れる。硬貨は重さがある。カードは少なくともプラスチックの板だ。でもブロックチェーン上のトークンには、物理的な実体がない。

　蓮は歩きながら考えた。

　UIだ。

　問題を解くのはUIだ。ユーザーインターフェース。技術をどう見せるか。いや、**技術を見せないこと**が、最良のUIなのかもしれない。

　根津がQRコード決済を使うとき、裏でPolygonのネットワークがトランザクションを処理していることを知る必要はない。ERC-20の`transfer`関数が呼ばれていることを知る必要はない。ガス代がどのバリデータに配分されるかなんて、絶対に知る必要がない。

　根津が知るべきことは一つだけだ。

　**ピッと鳴ったら、お金が届いた。**

　それだけでいい。それ以外は全部、蓮のコードが引き受ける。

　技術の最高到達点は、技術が見えなくなることだ。電気のスイッチを入れるとき、発電所の仕組みを考える人はいない。蛇口をひねるとき、浄水場のフィルターを想像する人はいない。優れたインフラは、透明になる。

　蓮が作るべきは、ブロックチェーン決済アプリではない。

　**根津が「ピッ」と鳴らせるだけの、シンプルな箱だ。**

　裏側でどれだけ複雑な技術が動いていようと、表に出すのは「金額を入力する」「QRコードを見せる」「ピッと鳴る」の三ステップだけ。それ以上の情報は、ノイズだ。ノイズは信頼を殺す。

　蓮の足は、いつの間にか自宅に向かっていた。頭の中で、画面設計が動き始めている。ねづ珈琲のカウンターに置くタブレット。大きなボタン。数字を入力するテンキー。金額を確定するとQRコードが表示される。客がスマホで読み取る。決済完了。

　根津でも使える。

　いや、根津**に**使わせてみたい。あのごつい指で、ぶっきらぼうにタブレットを叩いて、「……はえぇな」と呟く顔が見たい。

　だが今日、蓮はそれを見る権利を失った。根津に断られたのだ。「暗号資産？　悪いが、そういうのは間に合ってる」。

　説得する材料が足りない。技術の説明では通じなかった。蓮に必要なのは、根津の言葉で語れる何かだ。

　商店街の入口に戻ったとき、蓮の目にポスターが映った。

```
【J-Coin 開発者ミートアップ vol.12】
「日本円ステーブルコインのエコシステムを語ろう」

日時：7月12日（金）19:00〜21:00
場所：渋谷 LODGE（Yahoo! LODGEイベントスペース）
登壇：丘部典孝（ジェイコイン・ラボCEO）ほか
参加費：無料

#JCoinDev
```

　ポスターではなかった。スマートフォンの画面だった。Discordの`#events`チャンネルに投稿されたイベント告知を、蓮はいつの間にか開いていた。

　七月十二日。三日後。

　蓮は「参加する」ボタンを押した。

---

　三日後。

　渋谷のイベントスペースは、蓮が想像していたよりずっとカジュアルだった。

　SIer時代の「勉強会」といえば、企業のセミナールームでスーツを着た人間がパワーポイントを映す退屈な儀式だった。でもここは違う。広いラウンジにスタンディングテーブルが点在し、缶ビールとピザが無料で振る舞われ、五十人ほどの参加者がTシャツ姿で立ち話をしている。会場のスクリーンにはJ-Coinのロゴと、今日のアジェンダが映し出されていた。

　蓮は入口で立ち尽くした。知り合いがいない空間に飛び込む度胸は、SIerを辞めるのに使った分で底をついていた。

「初めてですか？」

　横から声をかけられた。振り返ると、蓮より少し年上に見える女性が紙コップのビールを持って立っていた。黒髪のショートカットに、鮮やかな青のパーカー。胸元にはラベルシールが貼ってある。「陳美華（MeiHua Chen）/ Cross-border Payments」。

「あ、はい。初参加で……」

「私もまだ三回目。大丈夫、最初はみんなそう」

　陳美華は流暢な日本語で笑った。アクセントがわずかにある。

「何を作ってるんですか？」

「あ——まだ作ってないんですけど。商店街向けのQRコード決済アプリを、J-CoinのSDKで」

「おー、面白い！　私はクロスボーダー送金。台湾の家族にお金送るのに銀行使うと、手数料が何千円もかかって。J-Coinなら数円でしょ。それでSDK使い始めたの」

　蓮は少し肩の力が抜けた。同じ動機だ。手数料という構造的な問題に、ブロックチェーンで挑もうとしている。自分だけじゃなかった。

「あ、始まるよ」

　美華がスクリーンを指さした。会場の照明が落ち、ステージに一人の男が上がった。

　丘部典孝。

　YouTubeの配信で見た顔だ。でも画面越しに見るのと、同じ空間で見るのでは印象が違った。背が高い。百八十センチ近くあるだろうか。でも姿勢に威圧感はない。マイクを持つ手が自然体で、目線が会場全体を柔らかく巡る。

「今日は来てくれてありがとうございます。ジェイコイン・ラボの丘部です」

　拍手。丘部は軽く頭を下げた。

「今日は技術の話をするつもりでしたが、予定を変更します。僕が今一番聞きたいのは、皆さんの話です」

　会場がざわついた。

「J-Coinを使って何を作ろうとしているのか。どんな問題を解こうとしているのか。僕らはSDKとAPIを提供しているけど、それを使って何が生まれるかは、皆さん次第です。今日は、皆さんの声を聞かせてください」

　マイクが参加者に回された。一人目はDeFiプロトコルを開発している若いエンジニア。二人目は法人間決済のスタートアップのCTO。三人目はNFTマーケットプレイスにJ-Coin決済を組み込もうとしている大学院生。

　蓮はマイクが近づくたびに心臓が跳ねた。自分の番が来たら何を言えばいい。ブロックチェーン歴一週間の人間が、この場で発言していいのか。DeFiもNFTも、蓮にはまだ概念すら曖昧な言葉だ。

　四人目。マイクが蓮の前を通り過ぎようとした。

「——すみません」

　声が出ていた。自分でも驚いた。マイクが差し出される。蓮はそれを両手で受け取った。汗で滑りそうだった。

「藤崎蓮です。フリーランスのWebエンジニアで、ブロックチェーンは完全に初心者です」

　会場に小さな笑いが起きた。好意的な笑いだ、と蓮は自分に言い聞かせた。

「商店街のQRコード決済アプリを作ろうとしています。行きつけの喫茶店のマスターが、キャッシュレス決済の手数料で苦しんでいて。三・五パーセント。年間三十万円。コーヒー一杯ずつ積み上げた利益が、構造的に吸い上げられている。J-CoinのSDKを使えば、手数料をほぼゼロにできるんじゃないかと」

　蓮は一息ついた。

「でも先日、そのマスターに提案したら断られました。『暗号資産は御免だ』って」

　会場が静かになった。丘部が腕を組んでいるのが見えた。その表情は読めない。

「技術的にはできると思うんです。でも問題はそこじゃなかった。マスターが信頼できる形で届けないと、どんなに優れた技術も意味がない。僕はまだ、その答えを持っていません」

　蓮はマイクを返した。指が震えていた。

　十秒ほどの沈黙。

「——面白い」

　丘部の声だった。

　会場が丘部を見た。丘部はステージの端に座り、片手で顎を撫でていた。

「藤崎さん、だっけ。ちょっとそのまま立ってて」

　蓮は棒立ちになった。

「商店街。QRコード決済。手数料問題。しかも、もう断られている」

　丘部は笑った。嘲笑ではなく、何かを見つけた人間の笑い。

「僕がジェイコイン・ラボを作ったとき、最初にやりたかったことがまさにそれなんだよ。ステーブルコインを『普通の人』が『普通に使える』ようにすること。でも僕らはインフラ企業だ。SDKは作れるけど、アプリは作らない。作るべきじゃないと思っている。なぜか分かる？」

　蓮は首を振った。

「インフラ企業が自分でアプリまで作ると、エコシステムが育たない。僕らが決済アプリを出したら、他の開発者は『じゃあ自分が作る必要ないな』と思ってしまう。でも僕らに商店街のおじさんの気持ちは分からない。毎日コーヒーを飲みに行って、手数料の愚痴を聞いて、その人の目の高さで技術を届けられるのは——藤崎さん、君のような開発者だけだ」

　蓮は息を呑んだ。

「うちはSDKしか出さない。でも、こういうアプリを作ってくれる開発者が、一番欲しかったんだ」

　丘部が立ち上がった。

「ちょっとこのあと、時間ある？　話の続きがしたい」

---

　懇親会の喧騒を抜けて、蓮と丘部は会場の隅のスタンディングテーブルに向かい合った。美華もいつの間にか隣に立っている。「面白そうだから混ざっていい？」と聞かれ、蓮に断る理由はなかった。

　丘部はビールを一口飲み、蓮にまっすぐ目を向けた。

「商店街のマスターに断られた理由、分かる？」

「……暗号資産だと思われたから、ですかね」

「それは表面の理由。本当の理由は、信頼の接点がないからだ」

　蓮は眉をひそめた。

「僕らがJ-Coinを法令準拠で発行しているのは、技術の正しさだけでは人はお金を預けてくれないからだよ。法律という社会の合意があって、初めて『これは安全だ』と思ってもらえる。君のマスターに必要なのも同じだ。技術の説明じゃない。『これは法律で守られた、ちゃんとした仕組みだ』という安心感」

「でも僕は法律の専門家じゃ——」

「だから、うちの白石を紹介する」

　丘部がスマートフォンを取り出した。

「白石楓。うちの法務責任者。金融庁出身で、ステーブルコインの規制に一番詳しい人間の一人だ。アプリ開発者が知っておくべき法的な整理は、彼女に聞くのが一番早い」

　蓮は目を瞬いた。「そこまでしてもらって、いいんですか」

「勘違いしないで。僕らが君の味方をするわけじゃない。君のアプリが成功すれば、J-Coinのエコシステムに実用的なユースケースが一つ増える。僕らにもメリットがある。Win-Winの関係だ」

　丘部はスマートフォンの画面を蓮に向けた。Slackのメッセージ作成画面。宛先には「白石楓」と入っている。

「それに、エコシステムの開発者をサポートするのは、インフラ企業の仕事だ。SDKのドキュメントを整備するのと同じで、開発者が法律でつまずかないようにするのも僕らの責任範囲だと思っている」

　蓮は丘部の目を見た。YouTubeの配信で見た穏やかな目。でもその奥に、鋼のような確信がある。この人は本気だ。ステーブルコインを日常に届けるということに、会社の存在を賭けている。

「丘部さん」

「ん？」

「根津さん——喫茶店のマスターを説得できたら、パイロット導入をしたいんです。でも僕は、アプリを作ることしかできない。法律の話も、ステーブルコインの安全性の話も、僕の言葉じゃ根津さんには届かない」

　丘部は黙って蓮を見ていた。

「そのとき——直接、話してもらえませんか。根津さんに」

「マスターに？」

「はい。J-Coinが何なのか。暗号資産とどう違うのか。なぜ安全だと言えるのか。それを、丘部さんの口から」

　丘部はビールのコップをテーブルに置いた。五秒ほど黙り、それから静かに笑った。

「君、営業もできるタイプだな」

「SIerでは議事録しか書けませんでした」

「議事録を書ける人間は、人の話を聞ける人間だ。それが一番の才能だよ」

　丘部は頷いた。

「分かった。ただし条件がある」

「条件？」

「まずアプリのプロトタイプを作れ。テストネットでいい。動くものを見せないと、僕が何を言ってもマスターには響かない。法律の話は白石が、ステーブルコインのビジョンは僕が話す。でも『このアプリで決済ができる』と見せるのは、藤崎さん、君の仕事だ」

　蓮の心臓が速くなった。

「——作ります」

「期限は？」
